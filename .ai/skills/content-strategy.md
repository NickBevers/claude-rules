# Content Strategy — Planning & Creation

Create high-quality content for agency self-promotion or client deliverables.

## Step 1: Gather Context

Content type (blog, case study, portfolio, landing page, newsletter), target audience/knowledge level, goal (thought leadership, lead gen, SEO, education), key message, tone, SEO keywords, length.

## Step 2: Content Brief (parallel agents)

**Agent A — "Structure"**: Outline with headline, subheadings, key points per section, CTA. Focus on flow and scanability.

**Agent B — "Hook"**: Opening angle, 5+ headline variants, unique perspective differentiating from existing content.

## Step 3: Content Types

**Blog Post**: Specific benefit-driven headline, hook in first 2 sentences, subheadings every 2-3 paragraphs, real working code examples, clear CTA (not "thanks for reading").

**Case Study**: Challenge > Approach > Solution > Results. Include metrics, timeline, tech stack, client quote (or placeholder), before/after, live link.

**Portfolio Piece**: Overview, role, tech stack with rationale, key challenges solved, results/impact, screenshots.

## Step 4: Draft & Review

Write full draft. Self-audit for scanability, paragraph value, CTA clarity. Present to user.

## Rules

- Write in user's voice, not generic AI copy
- No filler phrases ("In today's digital landscape...", "Let's dive in...")
- No emojis unless requested. Code examples must be real and working.
- SEO-aware but human-first — never keyword-stuff
