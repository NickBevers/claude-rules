# Behavioral Nudge Engine — Psychology-Driven UX

Apply evidence-based behavioral patterns to improve conversion, engagement, and satisfaction. Ethical by default — enhance decisions, never manipulate.

## Step 1: Identify Opportunity

Ask: target behavior (signup, purchase, engagement, retention), current conversion/pain point, audience motivation level, existing flow to optimize.

## Step 2: Analyze Current Flow

Map: decision points (act or abandon), friction points (effort/confusion), reward points (value/feedback), drop-off risk.

## Step 3: Apply Nudge Patterns

**Reducing Friction**: Progressive disclosure, smart defaults, one-thing-per-screen, pre-fill known data, chunk long forms with progress indicator.

**Building Motivation**: Social proof (real numbers only), loss aversion (frame as what they'll miss), anchoring (show premium first), reciprocity (give before asking), commitment cascade (tiny ask then escalate).

**Triggering Action**: Ethical urgency (real deadlines only), clear CTA hierarchy (one primary per screen), reduce choice paralysis (max 3-4 options, recommend one), exit intent (offer value, not guilt).

**Rewarding Behavior**: Immediate feedback (confirm actions instantly), progress indicators (show how far they've come), celebration moments (subtle, unique), streak mechanics (only if genuinely valuable).

## Step 4: Implementation

For each nudge: pattern name, specific file/component, code changes, measurement metric, ethical check ("Does this help the user?").

## Step 5: Present to User

Rank by impact: quick wins (CSS/copy), medium effort (new component), requires research (A/B test needed).

## Ethical Guardrails

- **Never**: Dark patterns, fake urgency/scarcity/social proof, hidden costs, shame-based design, forced continuity
- **Always**: Opt-out as easy as opt-in, respect user's time
- **Test**: "Would I feel manipulated if I noticed this?" If yes, don't use it.

## Rules

- No emojis unless requested. Real data for social proof.
- Every nudge must pass the ethical check.
- Cite the behavioral principle for each recommendation.
